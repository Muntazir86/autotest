# External Resources

This page collects articles, papers, videos, tutorials, and integrations about Autotest written by the community. These resources provide additional perspectives, use cases, and implementation examples.

## Integrations

- [Robot Framework AutotestLibrary](https://github.com/aaltat/robotframework-Autotest) by **@aaltat**

    - **Description**: Robot Framework integration with Autotest. Automatically generates test cases from OpenAPI/GraphQL schemas using DataDriver integration.

    - **Documentation**: [Keyword documentation](https://aaltat.github.io/robotframework-Autotest/AutotestLibrary.html)

    - **Installation**: `pip install robotframework-Autotestlibrary`

- [Deriving Semantics-Aware Fuzzers from Web API Schemas](https://ieeexplore.ieee.org/document/9793781) by **@Zac-HD** and **@Stranger6667**
